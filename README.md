# Slot Machine Project

Прототип слота, разработанный на PixiJS и TypeScript.

### Описание
Игровой движок слота, реализующий сетку 5x3, логику ставок, систему баланса и визуализацию выигрышных линий с использованием GSAP анимаций.

### Технологии
* **PixiJS** — рендеринг графики.
* **TypeScript** — строгая типизация.
* **GSAP** — анимации и игровые эффекты.

### Инструкция по запуску

 **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com/K1T0C/slot-machine.git](https://github.com/K1T0C/slot-machine.git)
   cd slot-machine
   Установите зависимости:
   npm install
   Запустите проект:
   npm run dev
   Откройте в браузере:
Перейдите по адресу, который укажет терминал,обычно(http://localhost:....)
